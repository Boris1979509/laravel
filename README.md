## Backpack laravel

#### Download repository:
```sh
git clone https://github.com/Boris1979509/laravel.git
```
#### Go to the app:
```sh
$ cd laravel
```
#### Install dependencies:
```sh
$ npm install && composer install
```
#### Run migrate with seeds:
```sh
$ php artisan migrate --seed
```
#### Run serve:
```sh
$ php artisan serve
```
